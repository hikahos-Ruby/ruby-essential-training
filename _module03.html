CONDITIONALS: IF, ELSE, ELSIF
    

        • Structure of an if statement:
        

        if boolean_condition
            a_set_of_instructions
        elsif another_boolean_condition
            another_set_of_instructions
        else
            default_set_of_instructions
        end
        

        • Concise if statement:   
        	>> name = "John" >> puts "This is true" if name == "John" returns => This is true    
    

    CONDITIONALS: UNLESS, CASE
    

        • Other conditionals:
        • unless (equal to if !)
        

        unless boolean_condition
            a_set_of_instructions
        end
        

        • case
        

        case
        when boolean_condition
        	a_set_of_instructions
        when another_boolean_condition
        	another_set_of_instructions
        else
        	default_set_of_instructions
        end
        

        • ternary operator (if condition is true, return first value, otherwise return second value)
        

        boolean_condition ? value_if_true : value_if_false
        

        • or operator (if variable has value, set it to that value, otherwise set it to another value)
        

        variable = value_if_true || value_if_false
        

        • or-equals operator (if variable has no value, set a default value to it)
        

        variable ||= default_value
        

    

    LOOPS
    

        • Structure of a loop:
        

        loop do
            a_set_of_instructions
            break if boolean_condition (exits loop)
            next if boolean_condition (skips to the next loop)
            another_set_of_instructions
        end
        

        

        • Loop controls:
        • break = terminate the whole loop
        • next = jump to the next loop
        • redo = redo the loop
        • retry = start the whole loop over  
        

        • Other types of loop:
        • while (similar logic to if)
        • until (similar logic to unless)
    

    ITERATORS
    

        • Iterators are loop methods
        • times
        

        x = 5
        x.times do |i|
            print i.to_s + ":" + x.to_s + ","            
        end
    
        returns => 0:5,1:5,2:5,3:5,4:5,
        

        • upto
        

        x = 1
        x.upto(5) do |i|
            print i.to_s + ":" + x.to_s + ","            
        end
    
        returns => 1:1,2:1,3:1,4:1,5:1,
        

        • downto
        

        x = 5
        x.downto(1) do |i|
            print i.to_s + ":" + x.to_s + ","            
        end
    
        returns => 5:5,4:5,3:5,2:5,1:5,
        

        • each
        

        x = 1..5
        x.each do |i|
            print i.to_s + ":" + x.to_s + ","            
        end
    
        returns => 1:1..5,2:1..5,3:1..5,4:1..5,5:1..5,
        

        

        • Iterators available for each object type:
        • Integers/floats: times, upto, downto, step
        • Range: each, step
        • String: each, each_line, each_byte
        • Array: each, each_index, each_with_index 
        • Hash: each, each_key, each_value, each_pair
