INTRODUCING RUBY: WHAT IS RUBY?
    

        • Object-oriented programming language
        • Created in Japan in 1995 by Yukihiro 'Matz' Matsumoto
        • Syntax like Perl, Python, and Smalltalk
        • Not a compiled language (like C++, Java, Visual Basic)
        • Interpreted language, requires Ruby interpreter
        • Whitespace independent
        • No semicolons
        • Ruby on Rails - web framework written in Ruby
        • Ruby language website: http://www.ruby-lang.org
    

    INSTALLING RUBY ON THE MAC
    

        • Terminal command to install: brew install ruby
        • Terminal command to get version: ruby -v returns ruby 2.5.1p57 (2018-03-29 revision 63029) [x86_64-darwin17]
        • Terminal command to get install path: which ruby returns /usr/local/bin/ruby
    

    INSTALLING RUBY ON WINDOWS
    

        • Command prompt command to install: C:\> choco install ruby
        • Or download the one-click installer from the Ruby site: https://www.ruby-lang.org/en/downloads/
    

    INTERACTIVE RUBY SHELL (IRB)
    

        • Allows us to interact with code in real time 
        • Works like a calculator 
        • Great for testing code
        • Terminal command to start irb: irb
        • To start a simple irb prompt: irb --simple-prompt
        • puts returns nil and adds a line return, while print returns a result without a line return
