WHAT IS A CODE BLOCK    
    

        • A code block is a set of instructions that can be executed as a unit, such as a module, a class, or a function        
        • Common methods that use blocks:
        

        • find
        • find/detect
        >> (1..10).find {|i| i % 3 == 0} 					 returns => 3
        • find_all/select
        >> (1..10).find_all {|i| i % 3 == 0} 				 returns => [3, 6, 9]
        • any?
        >> (1..10).any? {|i| i % 3 == 0} 					 returns => true
        • all?
        >> (1..10).all? {|i| i % 3 == 0} 					 returns => false
        • delete_if
        >> [*1..10].delete_if {|i| i % 3 == 0} 			 returns => [1, 2, 4, 5, 7, 8, 10]
        

        • merge - used for hashes only
        

        hash1 = {"a"=>111, "b"=>222}
        hash2 = {"b"=>333, "c"=>444}
        hash1.merge(hash2)
    
        returns => {"a"=>111, "b"=>333, "c"=>444}
        

        

        • collect/map - works best with arrays, hashes, ranges
        >> ['cat','dog','bird'].collect {|i| i.capitalize} 	 returns => ["Cat", "Dog", "Bird"]
        

        • sort - compares values using '<=>' operator; returned value is either -1 (less than), 0 (equal), or 1 (more than)
        >> [3,1,5,2,4].sort {|val1,val2| val1 <=> val2} 		 returns => [1, 2, 3, 4, 5]
        

        • inject - accumulator, storing values in memo object
        

        array = [*1..10]
        product = array.inject {|memo, n| memo * n}
        returns => 7257600
        

        

        • Return value types:
        • find/detect 	 => Object or nil
        • find_all/select 	 => Array
        • any? 			 => Boolean
        • all? 			 => Boolean
        • delete_if 		 => Array
        • merge 		 => Hash
        • collect/map 	 => Array
