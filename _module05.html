DEFINING AND CALLING METHODS    
    

        • Declaring a method:
        

        def longest_word
            words = ['apple','pear','banana','plum']
            l_word = words.inject do |memo,word|
                memo.length > word.length ? memo : word
            end
            puts l_word
        end

        >> longest_word 	 returns => banana
        

        

        • Common convention for methods:
        • Underscore between words, like variable names
        • Method names can have ? - useful for tests and booleans
        • Methods must be defined before calling them
    

    VARIABLE SCOPE IN METHODS    
    

        • Instance variable in a method:
        

        @words = ['apple','pear','banana','plum']
        def longest_word            
            l_word = @words.inject do |memo,word|
                memo.length > word.length ? memo : word
            end
            puts l_word
        end

        >> longest_word 	 returns => banana
        
        
    

    ARGUMENTS IN METHODS    
    

        • Arguments are a comma-separated list of values that are passed into methods
        • Parameters are the variables in the method signature
        

        • Common convention for arguments in methods:
        • Methods with arguments typically use parentheses
        • Methods without arguments typically do not
        • Parentheses are optional in both cases
        

        • Passing argument to the method:
        

        fruits = ['apple','pear','banana','plum']
        def longest_word(words)           
            l_word = words.inject do |memo,word|
                memo.length > word.length ? memo : word
            end
            puts l_word
        end

        >> longest_word(words) returns => banana
        

    

    ARGUMENT DEFAULT VALUES    
    
        
        • Default argument values in methods:        
        
            
        def longest_word(words=['pomegranate'])                      
            l_word = words.inject do |memo,word|
                memo.length > word.length ? memo : word
            end            
            puts l_word
        end

        >> longest_word returns => pomegranate
        

    

    RETURN VALUES    
    
        
        • Return values in methods:        
        
            
        def longest_word(words=['pomegranate'])                      
            return l_word = words.inject do |memo,word|
                memo.length > word.length ? memo : word
            end  
        end

        >> value = longest_word returns => "pomegranate"
        

        

        • keyword return doesn't have to be at the end and there can be more than one
        • only one value is returned from a method, unless the return value is an array
        
            
        def long_short(words=['pomegranate','fig'])                      
            long = words.inject do |memo,word|
                memo.length > word.length ? memo : word
            end  
            short = words.inject do |memo,word|
                memo.length < word.length ? memo : word
            end
            return [long, short]
        end

        >> value = longest_word returns => ["pomegranate", "fig"]
        
        
    

    OPERATORS ARE ALSO METHODS
